# cspt-unpacker

Browser-compatible distribution of the CSPT `Unpacker` component.

This repository is generated from `cspt/dist/unpacker` and is intended to be consumed by the VS Processing extension.

## Build From Source

```sh
cd ../cspt
npm run build:unpacker-dist
```

Set `CSPT_DIST_DIR=/path/to/dist-repo` to write somewhere other than the auto-detected sibling checkout.
